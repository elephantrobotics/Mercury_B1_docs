
# 运输与保存

## 1 物流运输要求


|      |    |
|  ----  | ----  |
| 温度  | 0°C~50°C |
| 相对湿度  | 20%~70% |
| 运输时的方向| 机器人头部朝上，双臂垂下 |
| 运输时的外部条件  | 外部使用木架固定，防止挤压 |


## 2 设备储藏
|      |    |
|  ----  | :----  |
| 温度  | 0°C~50°C |
| 相对湿度  | 20%~70% |
| 运输时的方向| 机器人头部朝上，双臂垂下 |
| 叠压要求  | 不可叠压 |
| 储藏环境| 室内 |
| 其他环境要求| - 远离灰尘、油烟、盐分、铁屑等。 <br>- 远离易燃性、腐蚀性液体与气体。<br>- 不得与水接触。<br>- 不传递冲击与振动等。<br>- 远离强电磁干扰源。|
